本工程实现任意给定个股的基本面分析

其特征如下：
1. 可以通过数据库或者访问本地数据文件来获取基本面数据
2. 获取的数据可选存为本地文档
3. 可以实现指定个股连续多年（默认是从2010年开始）的基本面对比分析
4. 可以实现多个股（一般是同行业）的横向对比分析
5. 能够自动生成评估报告并绘图表示出来

--------------------------------------------------------------------------------------
API说明：
提供3个API供用户调用
1. Init(id_list, MODE) #初始化配置，主要是配置目标个股和数据来源 
  id_ilst:个股ID列表    MODE:数据来源选择， 
  MODE = "SQL" -> 数据来源于数据库 
  MODE = "CSV" -> 数据来源于本地CSV文件

2. GetData(switch) #自动获取数据 
  switch：是否将获取的数据存储到本地？ 只能为"ON"或者"OFF"

3. Analyse()    #分析数据并输出评估报告，并绘图显示 

--------------------------------------------------------------------------------------
使用示例：
import UserApi
id_list = ['000651', '000333', '600690']
if __name__ =='__main__':
    UserApi.Init(id_list,'SQL') #初始化配置：采用数据库访问的形式获取id_list的基本面数据
    UserApi.GetData('OFF') #获取数据，并禁止保存到本地
    UserApi.Analyse() #对数据进行分析，并绘图
    
备注：
如果选择使用SQL模式获取数据，请如下配置（CSV模式不用配置）
1. 在/sys_config文件夹下兴建一个 “account.cfg”的文件
2. 依次输入你的mysql账号的配置信息（每条配置独占一行）
ip
user_name
password
database

比如：
localhost
usr1
123456
db1
